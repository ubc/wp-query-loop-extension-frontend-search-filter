# WP Query Block Extension - Search Filter

Add search filter in the frontend page that filters the posts returned from the query block.

Documentation available at [https://cms.ubc.ca/support/cms-manual/list-of-blocks/wp-query-block-extension-frontend-search-filter/](https://cms.ubc.ca/support/cms-manual/list-of-blocks/wp-query-block-extension-frontend-search-filter/).
